## 1. Namespaces

Use:

- `minecraft:` for vanilla
- `brewing:` for your mod
- `modid:` for other mods

---

## 2. Tag Categories

Recommended tag groups:

### Beverage Tags

- `brewing:beer`
- `brewing:spirit`
- `brewing:fermentable`
- `brewing:distillable`

### Ingredient Tags

- `brewing:grain`
- `brewing:fruit`
- `brewing:botanical`

### Container Tags

- `brewing:glass`
- `brewing:metal`
- `brewing:wood`

### Equipment Tags

- `brewing:fermentation_station`
- `brewing:distillation_station`
- `brewing:aging_station`

---

## 3. Tag Usage Rules

- Tags must be unique.
- Tags must be lowercase.
- Tags must not contain spaces.
- Tags must not contain uppercase letters.
